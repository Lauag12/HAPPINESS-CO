#🎭 HAPPINESS&Co - Agenda Cultural

---

## 📌 Descripción del proyecto

HAPPINESS&Co es una aplicación web orientada a la **organización, gestión y visualización de eventos culturales**.

Su objetivo es ofrecer a los usuarios una agenda **clara, accesible e intuitiva**, donde puedan consultar actividades como conciertos, exposiciones, cine y teatro.

El proyecto integra tanto el desarrollo **frontend (HTML + Tailwind CSS)** como la lógica de negocio en **Java** y la gestión de datos mediante **MySQL**, complementado con un sistema de sindicación **RSS**.

Este trabajo ha sido desarrollado como proyecto final del módulo de **Desarrollo de Aplicaciones Web (DAW)**.

---

## 🎯 Propósito del proyecto

El propósito principal es aplicar de forma **práctica e integrada** los conocimientos adquiridos durante el curso, abarcando distintas áreas del desarrollo web:

### 🧩 Tecnologías y áreas aplicadas

* Lenguajes de marcas (**HTML5, CSS3**)
* Frameworks de estilos (**Tailwind CSS**)
* Programación orientada a objetos (**Java**)
* Bases de datos relacionales (**MySQL**)
* Sindicación de contenidos (**RSS - XML**)
* Control de versiones (**Git y GitHub**)

### 💼 Enfoque profesional

Además, se busca simular un entorno real mediante:

* Organización estructurada del proyecto
* Documentación técnica continua
* Uso de commits para trazabilidad del desarrollo
* Aplicación de principios de **UX (experiencia de usuario)** y **UI (interfaz)**
