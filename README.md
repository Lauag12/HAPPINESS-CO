# HAPPINESS&Co - Agenda Cultural
## Descripción del proyecto
HAPPINESS&Co es una página web orientada a la gestión y visualización de eventos culturales. Surge con el objetivo de organizar de forma clara y accesible las actividades culturales disponibles en una ciudad, como conciertos, obras de teatro o exposiciones.
Este proyecto ha sido desarrollado como práctica final del módulo de Desarrollo de Aplicaciones Web (DAW), integrando múltiples tecnologías vistas durante el curso.
---
## Propósito del proyecto
El objetivo principal es aplicar de forma práctica los conocimientos adquiridos en:
- Lenguajes de marcas (HTML, CSS)
- Bases de datos relacionales (MySQL)
- Programación (Java)
- Sindicación de contenidos (RSS)
- Entornos de desarrollo (Git y GitHub)
Además, se busca simular un entorno laboral real, siguiendo buenas prácticas como:
- Organización del código
- Documentación continua
- Uso de commits diarios para tener control del seguimiento o avance del proyecto.
